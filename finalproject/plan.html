Site Plan: WeatherCheck

Site Name

WeatherCheck

This name was chosen because it clearly communicate the purpose of the website, to check the weather. It's simple to say and remember, good for branding.

The domain can be: weathercheck.org

Site Purpose

The purpose of the website will be provide users with fast and accurate weather information. It will provide real time weather condition, detailed 5-day forecast and a feature that will allow the user to compare the weather between two different cities. It will gather information from the user's location to display the current location weather too.

Scenarios (Target Audience Questions)

  • "I'm planning a hiking trip for this weekend. What will the temperature and humidity be like in the mountains for the next 5 days?"
  • "My family is trying to decide between a vacation in Lisbon or Rome next week. Which city will have better weather right now to help us choose?"

Color Scheme

  • #001429 Header and Footer background color and buttons
  • #000A14 All main body text
  • #2693FF Titles
  • #FAFAFA Background color
  • #FFAD05 Hihglights in the website

Typography

The chosen font is "Roboto"

  • Font Family: "Roboto"
  • Headings: Roboto Bold (700 weight)
  • Body text: Roboto Regular (400 weight)
